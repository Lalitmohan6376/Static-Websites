# 🏋️‍♂️ FITNESS CLUB WEBSITE

A modern and fully responsive Fitness Club Website built using **Flask**, **HTML5**, **CSS3**, and **Vanilla JavaScript**.

Designed to provide an engaging user experience for fitness enthusiasts, gym members, and personal training businesses.

---

# ✨ FEATURES

* 💪 Modern Fitness-Themed Design
* 📱 Fully Responsive Layout
* 🏋️ Fitness Programs & Classes
* 👨‍🏫 Professional Trainers Section
* 📅 Weekly Training Schedule
* 💳 Membership Plans
* 📞 Contact Page
* ⚡ Fast & Lightweight
* 🍶 Flask Template Rendering

---

# 🚀 TECHNOLOGIES USED

## Frontend

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript

## Backend

* 🐍 Python
* 🍶 Flask
